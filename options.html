Basic improvements:


The "%" mark where to put the comment contents. Default value is "[NOTE: %]". Other common patterns are "(%)", "(Translator's note: %)" and "(TN: %)". Leave blank to not show hidden comments.

Replaced words and expressions:

    Add "/" at the beginning to use regular expressions. By default it will replace replace whole words case-sensitive.

    Example replacements (click to copy to above):

    • Fancy ellipses: ... → …
    • Fancy quotes: /"([^"]+)"/g → “$1”
    • Fix broken characters: /([☆★♡]+)/g → {\fnArial Unicode MS}$1{\r}
